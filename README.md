## Usage

- Fork this repo, clone to your desktop, and install dependencies:
  ```sh
  npm install
  ```
- Start the server, and navigate to `localhost:1234`
  ```sh
  npm run start
  ```

- Lint the JavaScript
  ```sh
  npm run lint
  ```
